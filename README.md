# 🚗 안전 중심 스마트 주차 보조 시스템 (2025.09 ~ 2025.12)

초음파 센서와 인체 감지 센서(PIR)를 융합하여 주차 중 차량과 장애물 간의 거리를 측정하고, **보행자 감지 시 즉각적으로 경고 모드로 전환하여 안전사고를 예방하는 스마트 주차 보조 시스템입니다**.

단순한 거리 측정을 넘어, 하드웨어 센서의 노이즈를 소프트웨어 알고리즘(EMA 필터, 타임아웃 보정)으로 극복하고 인체 감지 이벤트를 최우선으로 처리하는 Fail-Safe 기반의 상태 머신 제어 로직을 직접 구현했습니다.

---

## 🔧 Tech Stack

### Hardware & MCU
- **SBC**: Raspberry Pi 4B
- **Sensors**: 초음파 센서 (HC-SR04), 인체 감지 센서 (PIR)
- **Modules & Actuators**: I2C LCD(16×2), TFT Display, RGB LED 모듈, 패시브 부저

### Software & Protocols
- **Languages**: C 
- **Protocols & Interface**: I2C, GPIO

## 🚀 주요 기능

### 1. 센서 퓨전 기반 실시간 주차 모니터링
- 초음파 센서를 활용하여 차량과 장애물 간의 거리를 측정하고, I2C LCD와 TFT 디스플레이에 주차 상태 및 거리 데이터를 실시간 텍스트 및 이미지로 시각화

### 2. 다중 임계값 기반 3단계 상태 판단 로직
- 거리 측정값에 따라 상태를 분류하여 LED와 부저를 통해 직관적 피드백 제공
  - **SAFE** (> 50cm): 초록색 LED 점등 (충돌 위험 없음)
  - **CAUTION** (20~50cm): 노란색 LED 점등 (주의 단계)
  - **WARNING** (≤ 20cm): 빨간색 LED 점멸 및 부저 알림 (위험 단계)
 
### 3. 보행자 안전 우선(Fail-Safe) 제어 시스템
- 주차 공간 내 인체(PIR) 감지 시, 진행 중인 주차 안내를 즉시 중단하고 **DANGER 모드**로 최우선 전환하여 강력한 시청각 경고(LCD 텍스트, LED 점멸 순환 및 부저 출력) 발생 [6].

## 🛠 Troubleshooting

### 1. 센서 노이즈 필터링 및 타임아웃 예외 처리를 통한 신뢰성 확보
**문제**: 초음파 센서(HC-SR04)의 측정 불안정성으로 인해 데이터가 순간적으로 크게 튀거나, 장애물 각도로 인해 ECHO 펄스가 수신되지 않아 시스템이 무한 루프에 빠지며 정지하는 문제가 발생했습니다
<br><br>
**해결**: ECHO 펄스가 30ms 이내에 수신되지 않을 경우 타임아웃(Timeout) 처리를 통해 이전의 정상 유효 거리값(`last_valid`)을 유지하도록 펌웨어 단에 방어 코드를 구현했습니다 . 또한 수집된 데이터에 EMA(지수 이동 평균) 필터를 적용하여 순간적인 측정 노이즈를 완화함으로써 시스템 중단을 막고 거리 데이터의 신뢰성을 극대화했습니다.

### 2. 최우선 인터럽트(DANGER) 판정 설계
**문제**: PIR 센서가 일시적인 움직임이나 미세한 노이즈에 반응하여, 주차 안내 도중 불필요하게 보행자 경고 모드(DANGER)로 전환되는 오작동 현상이 발생했습니다 .
<br><br>
**해결**: 순간적인 노이즈를 실제 상황으로 오판하지 않도록 매 루프마다 PIR 센서의 상태를 재확인하고, **'인체 감지 신호가 5초 이상 연속으로 유지될 때만 DANGER 모드로 확정'**하는 상태 유지 시간 필터링 로직을 적용했습니다. 이를 통해 일시적인 오류를 무시하고 실제 보행자가 있을 때만 강력한 경고를 발생시켜 제어의 안정성을 크게 향상시켰습니다 .
